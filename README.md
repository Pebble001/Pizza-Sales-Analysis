## Problem Statement

This dashboard helps the business understand their sales pattern to provide insights to address business challenges relating to:
* Daily Sales Trend
* Hourly Sales Trend
* Best selling Pizza orders
* Lowest selling Pizza Orders


### Steps followed 


- Step 1: Loaded the excel file into power BI for visualization.

- Step 2: Also since the profile will be opened only for 1000 rows by default selected "column profiling based on entire dataset".

- Step 3: Removed all errors & empty values in each column.
  
- Step 4: In the report view, under the view tab, the theme was selected.

  
- Step 5 : Total revenue,total orders,total pizza sold, average order value, average pizza per order were respectively represented by card visuals.
